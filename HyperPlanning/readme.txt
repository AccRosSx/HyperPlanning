Ce projet à été réalisé sous Qt et est destiné à une distribution windows
Le fichier .sql est à importer sur votre serveur (langage utilisé : MySQL)
Le dossier source contient le code du programme
Dans le dossier HyperPlanning\prebuild\realease vous trouverez un .exe
Ce dernier à été générer via Qt et peut ne pas être compatible
Nous vous invitons, si vous le pouvez à compiler vous-même le code source
Nous avons vérifié avec DependencyWalker, mais nous ne pouvons pas être sûr que toutes les DLL soient présentes.
