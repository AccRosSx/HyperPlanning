Ce projet à été réalisé sous Qt et est destiné à une distribution windows
Le fichier .sql est à importer sur votre serveur (langage utilisé : MySQL et fichier généré par phpMyAdmin)
Le dossier source contient le code du programme
Dans le dossier HyperPlanning\prebuild\realease vous trouverez un .exe accompagné des .dll
Ce dernier a été généré via Qt et peut ne pas être compatible
Nous vous invitons, si vous le pouvez à compiler vous-même le code source
Nous avons vérifié avec DependencyWalker, mais nous ne pouvons pas être sûrs que toutes les DLL soient présentes.  

----

Dans l'éventualité où le programme Qt ne serait pas compatible nous vous fournissons une version alternative à utiliser en console
